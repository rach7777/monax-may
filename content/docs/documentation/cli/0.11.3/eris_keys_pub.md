# eris keys pub

Returns a machine readable pubkey given an address.

## Synopsis

Returns a machine readable pubkey given an address.

Command is equivalent to [eris services exec keys "eris-keys pub --addr ADDR"]

```bash
eris keys pub ADDR
```
