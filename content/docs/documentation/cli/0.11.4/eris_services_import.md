# eris services import

Import a service definition file from IPFS.

## Synopsis

Import a service for your platform.

```bash
eris services import NAME HASH
```

## Examples

```bash
$ eris services import eth QmQ1LZYPNG4wSb9dojRicWCmM4gFLTPKFUhFnMTR3GKuA2
```
