# eris-client status

Eris-Client Status Returns The Current Status From A Chain.

## Usage

```bash
eris-client status
```

## Synopsis

eris-client status returns the current status from a chain.



## Options

```bash
  --node-addr string   set the eris-db node rpc server address (default respects $ERIS_CLIENT_NODE_ADDRESS) (default "tcp://127.0.0.1:46657")
```
