# eris-client

Eris-Client Interacts With A Running Eris Chain.

## Usage

```bash
eris-client
```

## Synopsis

Eris-client interacts with a running Eris chain.

Made with <3 by Eris Industries.

Complete documentation is available at https://monax.io/docs/documentation
