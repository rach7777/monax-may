(static) jsonNumbers(outputSpec, output) → {*}

Returns the output an object with a field 'this.values.name' for each named output param. The raw output is put in 'this.raw'. ADDITIONALLY: All BigNumbers in 'this.values' are converted to numbers, so if you're using numbers outside the range of javascript 'Number', errors will be had.
