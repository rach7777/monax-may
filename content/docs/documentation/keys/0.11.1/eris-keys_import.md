# eris-keys import

eris-keys import <priv key> | /path/to/keyfile | <key json>

## Synopsis

eris-keys import <priv key> | /path/to/keyfile | <key json>

```bash

```

## Options

```
  -h, --help[=false]: help for import
      --no-pass[=false]: don't use a password for this key
  -t, --type="ed25519,ripemd160": import a key
```
