# eris config edit

Edit a config for in an editor.

## Synopsis

Edit a config for in your default editor.

```bash
eris config edit
```

  -m, --machine="eris": machine name for docker-machine that is running VM
  -n, --num=1: container number
  -v, --verbose[=false]: verbose output
